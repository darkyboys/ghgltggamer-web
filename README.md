The entire site is built with C++ so you will need either the GNU G++ or LLVM Clang++ to build the site.

Tweak the source in these files:
- build.cpp (Defines the main building logic of pages)
- widgets.hh (Defines the reusable widgets)
- utils.hh (Tiny utility library)

Once done clone the repo.
```bash
https://github.com/darkyboys/ghgltggamer-src/
cd ghgltggamer-src
```

Then give the permission to the shell scripts
```bash
chmod +x build.sh
chmod +x clean.sh
```

Once done. Run build
```bash
./build.sh
```

And you will have the files of the entire project that if you want then you can host.
